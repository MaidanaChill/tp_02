1.2- ¿Para qué sirve la etiqueta ol? Dar un ejemplo en donde lo utilizarías

La etiqueta ol sirve para armar listas de forma numerica y alfabeticamente, lo usaria para armar un listado de estudiantes alfabeticamente

  1. andres beltran
  2. cesar canepa
  3. josefina perez

2.1- ¿qué es una url absoluta?

Una url absoluta es una direccion web que proporciona un protocolo, dominio, ruta y algunas veces el puerto, la cadena de consulta y el fragmento.

2.2- ¿qué es una url relativa?

una url relativa es que describe la ubicacion de un recurso en relacion con la pagina, se utiliza para enlazar recursos internos y simplificar los enlaces
